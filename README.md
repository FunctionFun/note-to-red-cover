# note-to-red-cover

这是一个用于把笔记导出为小红书封面图片的Obsidian插件。

## 修改声明

本项目基于 note-to-red (https://github.com/Yeban8090/note-to-red) 修改，原项目版权归原作者 Yeban8090 所有，遵循 MIT 许可证。

## 1. 插件概述

note-to-red-cover 是一款专为 Obsidian 设计的实用插件，能够将您的 Markdown 笔记一键转换为符合小红书平台风格的封面图片。该插件解决了用户在内容创作过程中需要手动设计封面的痛点，通过自动化的方式生成美观、专业的小红书封面图片，极大提升了内容创作效率。

主要价值：
- 节省封面设计时间，专注于内容创作
- 保持封面风格统一，提升品牌形象
- 支持高度自定义，满足个性化需求
- 与 Obsidian 无缝集成，提升工作流效率

## 2. 核心功能列表

- **笔记转封面图片**：将 Obsidian 中的 Markdown 笔记内容自动转换为小红书封面图片，支持多行文本、列表和代码块。
- **实时预览功能**：在 Obsidian 中实时查看生成的封面效果，支持页面导航和缩放。
- **多种字体选择**：内置多种中文字体（宋体、黑体、楷体等），支持自定义字体设置。
- **背景图片配置**：支持自定义背景图片，可调整缩放比例和位置。
- **水印设置**：支持文字和图片水印，可调整透明度和密度。
- **复制与下载**：支持一键复制到剪贴板或下载生成的图片。
- **导出标识**：自动标记已导出的笔记（笔记属性），便于管理。
- **小红书作者信息**：支持添加小红书昵称、账号和简介等作者信息。

## 3. 安装指南

### 下载插件压缩包
目前没有上架到Obsidian商店，可手动下载插件压缩包使用。

1. 下载最新版本的插件压缩包
2. 解压到OB仓库中的 `/.obsidian/plugins/` 目录
3. 重启 Obsidian
4. 在设置中启用插件

### 3.2 开发环境安装

如果您想参与插件开发，可以按照以下步骤设置开发环境：

```bash
# 克隆仓库
git clone https://github.com/functionfun/note-to-red-cover.git
cd note-to-red-cover

# 安装依赖
npm install

# 开发模式（自动重新构建）
npm run dev

# 构建生产版本
npm run build
```

## 4. 基本用法示例

### 4.1 打开小红书封面预览

**方法一：通过功能按钮**

1. 在 Obsidian 左侧边栏找到「小红书封面预览」图标
2. 点击该图标即可打开预览面板

**方法二：通过命令面板**

1. 按下 `Ctrl/Cmd + P` 打开命令面板
2. 输入「打开小红书图片预览」
3. 选择该命令即可打开预览面板

### 4.2 调整基本设置

1. 打开预览面板
2. 点击「设置」按钮
3. 调整以下基本设置：
   - 作者信息：填写小红书作者信息
   - 字体选择：选择适合的字体
   - 字号调整：拖动滑块调整字体大小
   - 背景设置：选择或上传背景图片

### 4.3 复制和下载图片

- **复制图片**：点击预览面板中的「复制」按钮，图片将自动复制到剪贴板
- **下载图片**：点击预览面板中的「下载」按钮，图片将保存到本地

## 5. 高级功能说明

### 5.1 背景图片高级配置

通过设置面板，您可以对背景图片进行精细调整：

1. 缩放比例：调整图片大小
2. 位置调整：拖动图片到合适位置
3. 透明度：调整背景图片透明度

### 5.2 水印设置

水印位置是随机生成的。

设置支持多种选项：

1. 水印开关
2. 水印文字
3. 水印图片，支持上传本地图片（和水印文字无法同时使用）
4. 水印透明度
5. 水印数量
6. 水印颜色

### 5.3 自定义页脚

您可以自定义图片页脚信息：

1. 作者名称
2. 作者账号
3. 个人简介

### 5.4 基于分割线的内容分割

可在设置中开启或关闭基于分割线分割内容。

插件支持使用水平分割线（`---`）来手动分割内容，每个分割区域将生成一张独立的图片：

```markdown
# 封面1
这是第一张封面的内容

---

# 封面2
这是第二张封面的内容

---

# 封面3
这是第三张封面的内容
```

## 6. 常见问题解答

### 6.1 为什么生成的图片内容被截断？

- 检查笔记内容是否过长，建议将长内容分成多个封面
- 调整字体大小，减小字号可以显示更多内容
- 检查模板设置，确保模板高度足够

### 6.2 如何添加自定义字体？

1. 在设置面板中找到「自定义字体」选项
2. 输入字体名称和字体族
3. 保存设置后即可在字体选择中使用

### 6.3 为什么背景图片无法正常显示？

- 检查图片URL是否正确
- 确保图片格式支持（建议使用JPG、PNG格式）
- 检查网络连接是否正常

### 6.4 生成图片的速度较慢怎么办？

- 关闭不必要的其他插件
- 减小笔记内容长度
- 降低图片质量设置

### 6.5 生成的图片可以直接用于小红书吗？

是的，生成的图片已经优化为小红书平台的尺寸和风格，可以直接上传使用。建议在上传前检查内容是否符合小红书平台的审核要求。
